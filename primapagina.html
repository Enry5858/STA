SONO ENRICO FALLACE E INIZIO A STUDIARE L'HTML
CHI HA INVENTATO QUESTO LINGUAGGIO?

HTML è stato inventato da Tim Berners-Lee.

QUANDO È STATO INVENTATO?

È stato inventato nel 1991.

QUALI SONO GLI SCOPI DI QUESTO LINGUAGGIO?

Lo scopo è strutturare i contenuti di una pagina web.

CHE COS'È IL WEB?

Il web è un sistema di pagine collegate accessibili via Internet.

CHE PROBLEMI RISOLVE IL WEB?

Risolve la condivisione globale delle informazioni.

QUAL È LA DIFFERENZA TRA WEB, HTML ED INTERNET?

Il web è un sistema di pagine, HTML è il linguaggio per crearle, Internet è la rete globale che le collega.
